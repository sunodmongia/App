Project Basics

Follow these simple steps to create your first project:

  1. Click "New Project" button in your dashboard
  2. Enter project details: name, description, and goals
  3. Choose a template (optional) from our library
  4. Set project visibility (public/private/team)
  5. Configure settings: due dates, priority, and notifications
  6. Click "Create Project" to get started

💡 Pro Tip: Use descriptive project names and add detailed descriptions to help team members understand the project goals.

Project Template Gallery

Keep your projects organized and easily accessible using multiple organization methods:

📁 Folders

Create custom folders in your dashboard and organize projects by department, client, or project type. Simply drag and drop projects into folders.

🏷️ Tags

Add custom tags to projects for flexible categorization. Use tags like "urgent", "client-work", "internal", or "Q1-2025" for easy filtering.

⭐ Favorites

Star important projects to keep them pinned at the top of your project list for quick access.

🎯 Status Labels

Use status labels like "Active", "On Hold", "Completed", "Planning", or "Archived" to track project lifecycle stages.

Task Management

Creating and assigning tasks is simple and intuitive:

1

Open project and click "Add Task"

Navigate to your project dashboard and click the prominent "Add Task" button

2

Enter task details

Add a clear title, detailed description, and any relevant context or requirements

3

Set due date and priority

Choose appropriate deadlines and mark priority level (High, Medium, Low)

4

Assign team members

Use the assignee dropdown to select one or multiple team members

💡 Pro tip: Use @mentions in task descriptions to notify specific team members and create automatic links to related tasks.

Team Collaboration

Inviting team members is straightforward and includes flexible permission management:

1

Navigate to Project Settings → Team Members

2

Click "Invite Members" and enter email addresses

3

Choose appropriate permission level for each member

Permission Levels:

Admin: Full project control including settings, member management, and deletion
Editor: Can create, edit, assign tasks and manage project content
Commenter: Can view project content and add comments or feedback
Viewer: Read-only access to view project content and progress
