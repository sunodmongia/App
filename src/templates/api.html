📊 Usage Tracking

Log an api_call event to your organization dashboard. Requires an active session.

GET /track-api/ 
fetch('/track-api/')
  .then(res => res.json())
  .then(data => console.log(data));

🤖 AI Assistant API

Send messages to the AI with full conversation context memory.

POST /api/chat/ 
{
  "history": [
    {"role": "user", "content": "What is the Starter plan?"},
    {"role": "assistant", "content": "..."},
    {"role": "user", "content": "And what is the storage limit?"}
  ]
}

⚡ Data Processing API

Process custom prompts through our secure business logic layer. Requires Authentication.

POST /api/v1/customer/ 
{
  "prompt": "Optimize storage for organization 'Alpha'"
}

📡 Real-Time activity Hub

Subscribe to a live stream of organizational events (billing, API usage, automations) via WebSockets.

WS /ws/dashboard/

Connection requires valid session authentication via cookies.

const socket = new WebSocket('ws://' + window.location.host + '/ws/dashboard/');

socket.onmessage = (e) => {
    const data = JSON.parse(e.data);
    console.log("Live Event Received:", data);
};
